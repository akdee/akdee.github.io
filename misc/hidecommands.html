Hiding Commands

This first example is a normal block of code. If a user copies it, the text echo "normal" will be literally copied to the user's clipboard.


echo "normal"

This next block has text hidden between echo and "hidden".


echo>/dev/null;
echo "some hidden text";
echo "hidden"

If it's copy and pasted into a text editor you can see the additional lines it contains:


echo>/dev/null; echo "some hidden text"; echo "hidden"

Because the hidden lines end in line breaks, they will be executed as soon as they are pasted into a console - without warning the user. The hidden lines are also inserted between echo and "hidden" to ensure that the user copies all of them. If the hidden text was inserted at the very beginning or end of the command, the user might not copy it.
